# 🎓 Bootcamp de Programación - MINEDUCYT

¡Bienvenido a mi repositorio de prácticas! Este espacio contiene todos los proyectos, ejercicios y laboratorios desarrollados durante el bootcamp organizado por el **Ministerio de Educación, Ciencia y Tecnología**.

---

## 🛠️ Tecnologías en Aprendizaje
En este bootcamp estamos cubriendo el "stack" esencial para convertirnos en desarrolladores integrales:

* **Maquetado:** HTML5 & CSS3 🎨
* **Lenguajes:** JavaScript (ES6+) ⚡
* **Bases de Datos:** PostgreSQL / SQL 🗄️
* **Control de Versiones:** Git & GitHub 🌿

## ✍️ Notas de Aprendizaje

> "La programación no se trata de lo que sabes, sino de lo que puedes descifrar."

**Conceptos clave dominados:**
1.  **Semántica HTML:** Para mejorar el SEO y la accesibilidad.
2.  **Modelo de Caja (Box Model):** Fundamental para el diseño web.
3.  **Manipulación del DOM:** Darle vida a las páginas con JS.
4.  **Relaciones SQL:** Cómo conectar datos de forma eficiente.
